# 術語保護清單 (.eveglyph/glossary.md)

> 列出本 workspace 中不可被 agent 改寫語義的術語。一行一個，可加說明。
> agent 編輯文件時必須保留這些術語的既有含義與寫法。

- EveGlyph-MD — 語義優先的模組化 Markdown 封裝協議 / a semantic-first modular Markdown protocol
- EveGlyph Editor — 支援 EveGlyph-MD 的本地優先 agent-native 工作台
- diff-review — agent 變更先以 git diff 呈現，由人類 Accept/Reject，再落地
- local-first — 本地優先（資料與運算留在使用者機器）
- frontmatter — 文件開頭的 type/status/tags 分類中繼資料（不是指令）
